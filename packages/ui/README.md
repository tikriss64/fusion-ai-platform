# @fusion/ui — Componentes compartidos

Biblioteca de componentes **shadcn/ui** común a todas las apps. Las tres apps
originales usan dependencias idénticas (Radix + Tailwind + shadcn), así que se
unifican aquí en vez de repetirlas cuatro veces.

## Contenido (cuando se implemente)

- Componentes shadcn/ui (button, dialog, table, form, sidebar…)
- Tema y tokens de Tailwind compartidos
- Utilidad `cn()` y helpers de estilo
